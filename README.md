# Getting started

- Clone this repo
- Install `unrar` package
    #### For MAC
    ```
        brew install unrar
    ```
    #### For Ubuntu
    ```
        sudo apt-get update
        sudo apt-get install unrar
    ```    
- `npm install nodemon -g` to run the project in local server
- `npm install` to install all required dependencies
- `npm run dev` to start the local server
